---
title: Deploying with Azure
---

If you're using Azure as your cloud service provider, you must do the following to set up your deployment:

| Prerequisite | Link | 
| ------------ | ---- |
| The BigAnimal portal acts as your identity provider at first sign-in, but you can set up your own afterward. | [Setting up your identity provider](/biganimal/latest/getting_started/identity_provider/)|
| Connect your Azure Marketplace account to BigAnimal. | [Setting up your Azure Marketplace account](/biganimal/latest/getting_started/02_azure_market_setup/) |
| Check the readiness of your Azure subscription before deploying. | [Preparing your Azure account](/biganimal/latest/getting_started/preparing_cloud_account/01_preparing_azure/) |
| Connect your Azure cloud to BigAnimal. | [Connecting your Azure cloud](/biganimal/latest/getting_started/02_connecting_to_your_cloud/connecting_azure/) |
