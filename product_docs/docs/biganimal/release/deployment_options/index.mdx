You have two deployment options with BigAnimal: 

- Deploy clusters in your own cloud account with a cloud service provider
- Deploy clusters in BigAnimal's cloud account

To learn about the pricing of these two deployment options, see [Pricing and billing](/biganimal/latest/pricing_and_billing/).

## Using your own cloud account

When deploying in your own cloud account, you need to set up your cloud service provider. You can use your AWS or Azure account. In this deployment model, you are responsible for your managing your cloud resources and connecting your cloud to BigAnimal. To set up your cloud service provider, see [Checking your cloud readiness](/biganimal/latest/getting_started/preparing_cloud_account/). 

The following features are available when using your own cloud account: 

| Feature | Details |
| ------- | ------- | 
| Single node and high availability clusters | Supported in PostgreSQL 11-15 and EDB Postgres Advanced Server 12-15 | 
| Extreme-high-availability clusters | Supported in EDB Postgres Advanced Server 14-15 and EDB Postgres Extended Server 14-15 |
| Regions | Supported regions depend on your cloud service provider. See [Supported regions](/biganimal/latest/overview/03a_region_support/#your-own-cloud-account) for more information. |
| Postgres extensions | Supports amcheck, pageinspect, pg_buffercache, pg_freespacemap, pgrowlocks, pgstattuple, and pg_visibility. |
| Apache Superset | Apache Superset is supported. See [Analyzing your data with Apache Superset](/biganimal/latest/using_cluster/06_analyze_with_superset/) for more information. |

## Using BigAnimal's cloud account

BigAnimal's cloud account offers a seamless deployment option if you don't want to set up a separate cloud account for your clusters. You can deploy a cluster in BigAnimal's cloud account instantly. 

!!! Note Note
Currently, when you deploy in BigAnimal's cloud account, you can use only AWS as your cloud provider.
!!!

The following features are available when using BigAnimal's cloud account:

| Feature | Details | 
| ------- | ------- | 
| Single node and high availability clusters | Supported on PostgreSQL 11-15 and EDB Postgres Advanced Server 12-15. | 
| Postgres extensions |  Supports amcheck, dblink, pageinspect, pg_stat_statements, pg_surgery, pg_walinspect, and pgstattuple. |
| Regions | Certain regions are supported depending on your cloud service provider. See [Supported regions](/biganimal/latest/overview/03a_region_support/#your-own-cloud-account) for more information. | 
| Cloud infrastructure costs | BigAnimal bills you for cloud infrastructure costs. See [Cloud infrastructure costs](/biganimal/latest/pricing_and_billing/#biganimals-cloud-account) for more information. |
